## ADDED Requirements

### Requirement: 用户可以绑定本地文件夹作为视频源
系统允许用户添加、编辑、删除绑定的文件夹路径，配置持久化存储。

#### Scenario: 添加新的文件夹绑定
- **WHEN** 用户点击"绑定文件夹"按钮并选择有效路径
- **THEN** 系统保存路径到配置，开始后台扫描

#### Scenario: 验证文件夹存在性
- **WHEN** 用户输入的路径不存在或无访问权限
- **THEN** 系统显示错误提示，拒绝保存

#### Scenario: 删除已有绑定
- **WHEN** 用户点击删除按钮确认移除某个绑定
- **THEN** 系统删除配置，但保留数据库中的历史元数据

#### Scenario: 防止重复绑定
- **WHEN** 用户尝试添加已存在的路径
- **THEN** 系统提示"该路径已绑定"，不重复保存

### Requirement: 绑定配置持久化存储
系统使用 JSON 文件存储绑定配置，支持多路径列表。

#### Scenario: 应用启动时加载配置
- **WHEN** 应用启动
- **THEN** 系统从 `data/config.json` 读取绑定列表，展示在设置界面

#### Scenario: 配置自动保存
- **WHEN** 用户添加、编辑、删除绑定
- **THEN** 系统立即写入配置文件，确保重启后不丢失
