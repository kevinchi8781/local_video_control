## Why

本地视频文件缺乏统一的可视化管理工具，用户需要手动浏览文件夹层级、依赖系统播放器，无法快速预览和检索已分类的视频内容。本项目提供一个专用的视频面板管理器，通过树形导航 + 网格预览的方式，让用户能够高效浏览、搜索和播放本地视频。

## What Changes

- 新增本地视频扫描服务，递归遍历绑定文件夹，提取视频元数据（文件名、大小、时长、路径）
- 新增文件夹树形结构 API，返回层级分类供前端渲染左侧导航
- 新增视频列表 API，支持分页查询、按分类过滤
- 新增缩略图生成服务，使用 ffmpeg 截取视频关键帧作为封面
- 新增 React + Vite 前端界面，支持文件夹绑定、树形导航、视频网格浏览、内嵌播放
- 新增观看历史记录功能，支持继续观看和最近播放列表

## Capabilities

### New Capabilities

- `folder-binding`: 绑定本地文件夹路径，支持多个绑定源，配置持久化
- `video-scanning`: 递归扫描文件夹，识别视频文件（mp4、mkv、avi、webm 等），提取元数据
- `thumbnail-generation`: 使用 ffmpeg 截取视频关键帧生成 JPEG 缩略图
- `tree-navigation`: 生成文件夹树形结构 API，支持懒加载子节点
- `video-playback`: 视频流式传输，支持浏览器内嵌播放、播放进度记录
- `search-filter`: 支持按文件名搜索、按分类过滤、按时长筛选

## Impact

- **后端**: Node.js + Express 服务，新增文件扫描、元数据提取、视频流接口
- **前端**: React + Vite 单页应用，依赖 @ffmpeg/ffmpeg 或后端缩略图服务
- **系统依赖**: 需要安装 ffmpeg 用于缩略图生成和视频时长提取
- **存储**: SQLite 或 JSON 文件存储绑定配置、观看历史、元数据缓存
